第1个月英语朗读收获

发表于 2017-09-17 | 阅读量 次 | 分类于 英语朗读 | 阅读次数

此篇为《陪你读英语不出局》作业：
每月写一篇朗读收获的文章，发布简书http://www.jianshu.com/p/ebd135b76d0f
并投稿至“陪伴读英语不出局”

我在8月24号加入知识星球的《陪你读英语不出局》。在社群帮助下很快找到了我的语伴@李飞。快一月了，目前发表了24次的录音，出勤率在90%以上吧—-只有几天没打卡。这周打卡全勤，给自己点赞！

这里总结一下第1个月加入英语的收获：

初期加入社群的目的，是为了多培养一个习惯（之前获得的习惯是健身），现在看来，每天朗读英语—-通过行动我做到了。

刚开始的不顺心

最开始我尝试在早上或下班后朗读，刚开始还未形成朗读的安排和习惯，对自己的发音不习惯，不满意，生词多，自然是不顺心的。不过谁又能一开始就能做好呢。

慢慢摸索门道

随着朗读次数的增加，渐渐地放开敢讲了，也慢慢摸索到让自己朗读的门道：

每天7点前起床，刷牙洗脸，做一组简短的晨间操和冥想（早起早操和冥想是我在刻意练习的新习惯，也是在摸索），大概7点50开始：用电脑chrome打开sougou_weixin的清晨朗读的公众号的页面，最近文章即是今天要朗读的文章，我发现这样打开文方式比公众号会翻文章快多了。

接着，用手机打开喜马拉雅收听清晨朗读会的音频（可以调速），按照Jone的建议：

  1. 不看文章，先用正常音速听一遍，只能听个大概。
  2. 然后0.75倍速跟着朗读一遍。期间略过生词和不顺的地方，保持连惯
  3. 继续用0.75倍速，再跟读一次，这次有生词或读不顺的地方，暂停查单词或这句话多读几遍
  4. 再重复一次第2步，或自己朗读一遍
  5. 录音上传打卡

这样整体朗读4-5遍，大概20分钟左右。不过最终效果可能仍不太理想，比如有些生词发间卡住，或句子不顺，但我也是选择接受，因为追求完美需要投入更更多的时间和精力。其实也不用担心这次做的不好，因为以后会变好的，只是需要时间和更多的练习次数。

我想我已经做到了“快速入门”，现在朗读没压力，接着继续实践就可以了。
